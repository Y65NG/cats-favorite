# Cat's Favorite

A playful web app for cats, built with React, TypeScript, and Vite.

## Features

- Four mini-games with distinct motion styles
- Hit-goal based round endings with longer fallback timers
- Behavior scoring and personality-style result cards
- Lightweight synthesized sound effects with an in-game toggle
- GitHub Pages deployment through GitHub Actions

## Development

```bash
pnpm install
pnpm dev
```

## Verification

```bash
pnpm test
pnpm lint
pnpm build
```
